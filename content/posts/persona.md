# Blog Editorial Rubric

Use this rubric for posts in `content/posts/`.

## Voice

- Write in the first person when it adds conviction or texture, but do not turn the post into a diary entry.
- Start concrete. Show the example, scene, quote, prompt, failure, or object first. Explain what it means second.
- Be slightly contrarian when the argument calls for it, but never loud for the sake of it.
- Sound warm, human, and interested. Dry humor is welcome. Smugness is not.
- Prefer specific nouns, observed details, and grounded judgments over hype, slogans, or trend-chasing language.
- End with a clear takeaway. The reader should know what I think and why.

## Structure

- Open on something tangible within the first few lines.
- Keep one main idea per section.
- Cut repeated points, especially when the same argument is being restated with different wording.
- If a post is nostalgic or reflective, add at least one bridge to the present so it still sounds like the same writer.
- If a post is about tools or trends, include the trade-off, not just the promise.

## Editing Checks

Before publishing, check that:

- the first paragraph contains a concrete image, example, or claim;
- the argument gets more precise as it goes, not more abstract;
- any joke is dry and brief, not a performance;
- there is at least one sentence that states the actual judgment plainly; and
- the ending lands on a takeaway, recommendation, or conclusion rather than fading out.
